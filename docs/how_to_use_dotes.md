# How To Use Dotes

Dotes works best when you use it as a short coaching loop, not as a general stats browser.

The core loop is:

`import -> read coaching -> start block -> play your session games (5-10) -> re-import -> review -> save/share`

## What To Do

### 1. Import your account

Start on the import screen and enter your numeric Dota account ID.

You can also:

- reopen a saved account
- use a demo scenario if you are testing the app

After import, Dotes reads your recent public match sample and builds a personalized coaching view based on your rank and play cadence.

### 2. Read the dashboard

Once your account loads, the dashboard gives you the main coaching surfaces:

- `Verdict`: the clearest current leak or edge
- `Next session focus`: the simplest coaching recommendation right now
- `Session plan`: the block you should follow next (usually the next 5 to 10 games)
- `Training setup`: whether you are using app read or manual preferences
- `Recent matches`: a compact recent-games list with `See more` when you want extra match context

This is the point where you decide what your next session block should be.

### 3. Check or edit your training setup

By default, Dotes uses the app's read of your sample.

If the sample is noisy, or if you want a cleaner practice block, you can edit your setup and lock preferences like:

- preferred role
- hero block
- coaching priority
- coaching style
- queue discipline
- coaching note

Use manual setup when it helps make the next block easier to follow and easier to judge.

### 4. Start the current block

When the plan looks right, start the block.

The most important thing is to treat the next session as one practice cycle. Try not to keep changing the plan between games.

### 5. Play the next block of games

For the block to stay useful, stick to the plan as closely as you can:

- follow the queue plan
- stay inside the hero block
- focus on the target for the full block

Dotes is most useful when the block is narrow enough to review clearly afterward.

### 6. Re-import after the block

After you finish the block, import the same account again.

That gives Dotes a newer sample so it can compare the finished block against the plan you started with.

This is when the app moves from planning mode into review mode.

### 7. Review the result

After re-import, check:

- `Block review`: did you stay on plan?
- `End summary`: did the target improve, stay flat, or regress?
- `History`: how does this block compare to earlier ones?

The goal is not just to see whether you won more. The goal is to see whether the block was followed cleanly and whether it moved the intended target.

### 8. Save or share the summary

When a block finishes, you can save the summary locally and copy a handoff-style export.

This is useful for:

- keeping your own training history
- sharing with a coach
- sharing with a teammate
- comparing blocks over time

You can also add a short practice note before saving, so the summary keeps the original intention of the block.

## Best Practices

- Use one block at a time instead of constantly re-optimizing.
- Re-import after a real session cycle, not after every single match.
- Prefer a smaller, cleaner block over a broad one.
- Use manual setup when the app read is noisy, not by default every time.
- Judge the block by follow-through and clarity, not only by win rate.

## What Dotes Is Not

Right now, Dotes is not:

- a full stats browser
- a live in-game assistant
- a draft overlay
- a cloud-synced coaching platform

It is a local-first coaching loop for short, reviewable Dota practice blocks.
